Link Your Discord Account

  1. Open Discord and go to the server where the bot is active.
  2. Type /link name:YourMinecraftName in any allowed channel.
  3. The bot will reply with a verification code (e.g. ABCD-EF12).
  4. Join the Minecraft server as that player and type in chat:
    !link ABCD-EF12
  5. You'll receive a confirmation in both Discord and Minecraft.

To unlink, use /unlink in Discord. Admins can also unlink from the player profile.

Link Your Panel Account

  1. Sign in to the Minecraft Manager panel.
  2. Click your name in the top-right corner and select View Profile.
  3. In the Minecraft Account section, enter your Minecraft player name and click Link Account.
  4. You'll receive a verification code (e.g. ABCD-EF12). Join the Minecraft server as that player and type in chat:
    !link ABCD-EF12
  5. The panel will confirm the link automatically.

To unlink, go to your profile and click Unlink in the Minecraft Account section. Admins can also manage links from the identity settings.
